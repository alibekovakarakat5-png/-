# Инструкция по установке Node.js

## Проблема
Ошибка `npm : Имя "npm" не распознано` означает, что Node.js не установлен в системе.

## Решение

### Шаг 1: Установка Node.js

1. **Скачайте Node.js:**
   - Перейдите на официальный сайт: https://nodejs.org/
   - Скачайте **LTS версию** (рекомендуется, например, v20.x.x)
   - Выберите установщик для Windows (.msi файл)

2. **Установите Node.js:**
   - Запустите скачанный .msi файл
   - Следуйте инструкциям установщика
   - **ВАЖНО:** Убедитесь, что отмечена опция "Add to PATH" (добавить в PATH)
   - Завершите установку

3. **Проверьте установку:**
   - Закройте и откройте PowerShell заново (или перезагрузите компьютер)
   - Выполните команды:
     ```powershell
     node --version
     npm --version
     ```
   - Должны отобразиться версии Node.js и npm

### Шаг 2: Установка зависимостей проекта

После установки Node.js выполните:

```powershell
cd "C:\Users\USER\Desktop\AI CRM Analytics"
npm install
```

### Шаг 3: Запуск проекта

```powershell
npm run dev
```

## Альтернативный способ (если установка не помогла)

Если после установки Node.js команды всё ещё не работают:

1. **Проверьте PATH вручную:**
   - Откройте "Переменные среды" в Windows
   - Убедитесь, что в PATH есть путь к Node.js (обычно `C:\Program Files\nodejs\`)

2. **Перезапустите PowerShell** с правами администратора

3. **Или используйте Chocolatey:**
   ```powershell
   choco install nodejs
   ```

## Проверка версий

После установки должны работать команды:
- `node --version` - версия Node.js (должна быть v18+ или v20+)
- `npm --version` - версия npm (должна быть v9+)

## Дополнительная информация

- **Минимальные требования:** Node.js 18+ и npm 9+
- **Рекомендуется:** Node.js 20 LTS
- Если у вас уже установлен Node.js, но команды не работают, возможно нужно перезапустить терминал или добавить Node.js в PATH вручную
